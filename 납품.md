


- 납품 관리

	- 납품사 정보
		- 전체 기능 - 납품사 목록, 등록, 엑셀 다운로드
		- 납품사 정보
			- 필수 - 납품처 이름, 수정일자, 수정자
			- 납품 완제품 → 완제품 정보와 연결
			- 납품 반제품 → 반제품 정보와 연결
			- 선택 - 주소, 연락처, 담당자, 담당자 연락처, 담당자 email
		- 목록
			- 검색 조건
				- PC - 납품사 이름
				- Mobile - 납품사 이름
			- 목록 노출 항목
				- PC - 납품처 이름, 주소, 담당자, 담당자 연락처, 기능 버튼들
				- Mobile - 납품처 이름, 담당자 연락처, 기능 버튼들
			- 목록 기능
				- 상세 보기 - 납품처 이름에 링크 걸기
				- 수정 버튼 - 모든 정보 수정 가능
				- 삭제 - 납품사 정보 삭제 → 해당 정보를 사용하는 곳이 있을 경우 삭제 불가

	- 납품사 연계 업체 정보
		- 전체 기능 - 납품사 연계 업체 목록, 등록, 엑셀 다운로드
		- 납품사 연계 업체 정보
			- 필수 - 연계사 이름, 납품 업체 (납품사와 연결)
			- 선택 - 주소, 연락처, 담당자, 담당자 연락처, 담당자 email
		- 목록
			- 검색 조건
				- PC - 납품사 연계사 이름, 납품사
				- Mobile - 납품사 이름
			- 목록 노출 항목
				- PC - 납품처 연계 업체 이름, 납품사 이름, 주소, 담당자, 담당자 연락처, 기능 버튼들
				- Mobile - 납품처 연계 업체 이름, 담당자 연락처, 기능 버튼들
			- 목록 기능
				- 상세 보기 - 납품처 연계 업체 이름에 링크 걸기
				- 수정 버튼 - 모든 정보 수정 가능
				- 삭제 - 납품사 연계 업체 삭제 → 해당 정보를 사용하는 곳이 있을 경우 삭제 불가

	- 납품사 창고 정보
		- 전체 기능 - 목록, 등록, 엑셀 다운로드
		- 납품사 창고 정보
			- 필수 - 보유 납품사, 창고 이름, 주소 (주소 검색 적용), 수정일자, 수정자
			- 선택 - 보관 완제품 목록
		- 목록
			- 검색 조건
				- PC - 납품사 이름, 창고 이름
				- Mobile - 창고 이름
			- 목록 노출 항목
				- 보유 납품사, 창고 이름, 주소, 수정일자, 수정자
			- 목록 기능
				- 상세 보기 - 창고 이름에 링크
				- 수정 버튼 - 모든 항목 수정 가능
				- 삭제 버튼 - 창고 삭제

	- 완제품 정보
		- 전체 기능 - 완제품 목록, 등록, 엑셀 다운로드
		- 완제품 정보
			- 필수 - 완제품 이름, 완제품 코드, 완성차 회사 코드, 차량 코드, 차량 부위 정보, 색상 정보, 수정일자, 수정자
			- ig-config-manager 연동 코드 (모두 CAR_MAKER 하위 코드, API 드롭다운)
				- CAR_MAKER: 완성차 회사 코드 (현대, 기아 등)
				- VEHICLE_CODE: 차량 코드 - 14건 (CN7, DN8, GN7, NX4, MX5, LX2, BD, DL3, GL3, NQ5, MQ4, KA4, CV, MV)
				- PART_CODE: 차량 부위 - 17건 (범퍼, 펜더, 후드, 트렁크, 도어, 사이드스커트, 루프, 미러 등)
				- COLOR_CODE: 색상 - 15건 (크리미 화이트, 오로라 블랙펄, 그라비티 그레이, 인디고 블루 등)
			- 선택 - 두께, 폭, 길이
		- 목록
			- 검색 조건
				- PC - 완제품 이름, 완제품 코드
				- Mobile - 완제품 이름, 완제품 코드
			- 목록 노출 항목
				- PC - 완제품 이름, 완제품 코드, 차량 이름, 차량 부위 이름, 색상 정보, 기능 버튼
				- Mobile - 완제품 이름, 완제품 코드, 기능 버튼
			- 목록 기능
				- 상세 보기 - 완제품 이름에 링크
				- 수정 버튼 - 모든 항목 수정 가능
				- 삭제 버튼 - 완제품 정보 삭제

	- 반제품 정보
		- 전체 기능 - 반제품 목록, 등록, 엑셀 다운로드
		- 반제품 정보
			- 필수 - 반제품 이름, 반제품 코드, 색상 정보 (ig-config-manager COLOR_CODE 연동 드롭다운), 수정일자, 수정자
			- 선택 - 두께, 폭, 길이
		- 목록
			- 검색 조건
				- PC - 반제품 이름, 반제품 코드
				- Mobile - 반제품 이름, 반제품 코드
			- 목록 노출 항목
				- PC - 반제품 이름, 반제품 코드, 색상 정보, 두께, 폭, 길이, 기능 버튼
				- Mobile - 반제품 이름, 반제품 코드, 기능 버튼
			- 목록 기능
				- 상세 보기 - 반제품 이름에 링크
				- 수정 버튼 - 모든 항목 수정 가능
				- 삭제 버튼 - 반제품 정보 삭제

	- 완제품 입고요청/납품 관리
		- 전체 기능 - 제품 납품 요청 목록, 제품 납품 요청 등록, 엑셀 다운로드
		- 제품 납품 요청 등록
			- 필수 - 납품사, 납품 요청일, 납품 희망일
			- 완제품 정보 - N개, 반제품 정보 - N개 (납품사 납품 상품 종류에 따라 노출)
				- 완제품/반제품 상품
				- 수량
				- 납품 상태 (등록 시에는 노출하지 않음)
				- 버튼: 납품 완료, 반품 (등록 시에는 노출하지 않음)
		- 목록
			- 검색 조건
				- PC - 납품사
				- Mobile - 납품사
			- 목록 노출 항목
				- PC - 납품사, 납품 요청일, 납품 희망일, 완제품 종류 개수, 기능 버튼들
				- Mobile - 납품사, 납품 희망일, 기능 버튼들
			- 목록 기능
				- 상세 보기 - 납품 희망일에 링크
				- 수정 버튼 - 모든 항목 수정 가능
				- 납품 요청 취소 - 완제품 납품 상태가 모두 준비 중일 경우만 취소 가능. 한 개라도 납품 완료가 된 경우에는 취소 불가
		- 상태 값
			- 납품 요청 상태값
				- 납품 요청 - 상세 완제품들 상태가 모두 납품 요청인 경우
				- 부분 납품/반품 - 상세 완제품들 중 일부만 납품 or 반품이 된 경우
				- 전체 납품 - 상세 완제품들 상태가 모두 납품인 경우
				- 전체 반품 - 상세 완제품들 상태가 모두 반품인 경우
				- 납품 요청 취소 - 납품 요청 자체를 취소한 경우
			- 상세 완제품 납품 상태
				- 납품 요청 - 납품 요청 중인 상태
				- 납품 - 납품 완료된 상태
				- 반품 - 반품 처리된 상태
				- 납품 취소 - 납품 요청 취소 시 개별 완제품도 납품 취소 처리
