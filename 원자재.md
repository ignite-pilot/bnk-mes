


-  원자재 관리
	- 원자재 정보
		- 전체 기능 - 원자재 목록, 등록, 리스트, 엑셀 다운로드
		- 목록
			- 검색 조건
				- 원자재 종류, 원자재 이름
			- 항목
                필수 - 원자재 종류, 원자재 이름 (중복 불가), 등록일자, 등록자
                선택 - 색상, 두께, 폭, 길이, 원자재 업체 안전재고 수량, 비엔케이 창고 안전재고 수량, 수정일자, 수정자
			- 목록 기능
				- 상세 보기 - 원자재 이름에 보기 링크
				- 수정버튼 - 원자재 정보 수정. 원자재 이름, 안전재고 수량만 수정 가능
				- 삭제 - 원자재 정보 삭제 -> 삭제

- 원자재 공급 업체
	- 전체 기능 - 공급 업체 목록, 등록, 엑셀 다운로드
	- 공급 업체 정보
		- 필수 - 업체 명, 주소, 수정일자, 수정자
		- 선태 - 업체 연락처, 담당자, 담당자 연락처, 입고 요청 후 리드 타임, 발주 요청 후 리드 타임수정일자, 수정자
			- 제공 원자재 N개 -> 원자재 정보에서 선택
	- 목록
		- 검색 조건
			- 공급 업체 이름
		- 목록 노출 항목
			- 업체 명, 업체 연락처, 담당자, 담당자 연락처, 취급 원자재 개수, 수정일자, 수정자
		- 목록 기능
			- 상세 보기 - 업체 명에 링크
			- 수정 버튼 - 수정 가능항목 -> 업체 명, 주소, 업체 연락처, 담당자, 담당자 연락처, 입고 요청 후 리드 타임, 발주 요청 후 리드 타임수정일자, 제공 원자재 N개
			- 삭제 버튼 - 공급 업체 삭제. 

- 원자재 업체 창고 정보
	- 전체 기능 - 목록, 등록, 엑셀 다운로드
	- 업체 창고 정보
		- 필수 - 원자채 공급 업체 (원자재 공급 업체와 연결), 창고 이름, 주소 (주소 검색 적용), 수정일자, 수정자
		- 선택 - 보관 원자래 N개 -> 원자재 정보와 연결
	- 목록 
		- 검색 조건
			- 원자재 공급 업체, 창고 이름
		- 목록 노출 항목
			- 원자채 공급 업체 , 창고 이름, 주소, 수정일자, 수정자
		- 목록 기능
			- 상세 보기 - 창고 이름에 링크
			- 수정 버튼 - 수정 가능 항목 -> 원자채 공급 업체 , 창고 이름, 주소
			- 삭제 버튼 - 창고 삭제
