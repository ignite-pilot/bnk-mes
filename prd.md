# BNS를 위한 생산 관리 시스템
- 이 프로젝트는 생산 관리 시스템이야
- 모든 작업은 Sequential Thinking을 사용해서 수행해 주세요.
- 디자인 컨셉은 Simple한 디자인으로 해 주세요.
- 회원 연동은 ig-member project를 연동해줘. github에 read.me 파일을 보면 돼
* 개발 Guide는 CommonWebDevGuide.md 파일을 참고해줘
- 전체적인 기능
  - 아래 메뉴 구성 (클릭 하면 빈 페이지 연결)을 해주고, 회원 기능만 구축
  - 화면 구성은 상위에 대메뉴, 왼쪽에 상세 메뉴가 나오도록 해주고, Footer 영역도 만들어 줘. 
- 메뉴 구성
	-  원자재 관리
		- 원자재 정보
		- 원자재 공급 업체
		- 원자재 업체 창고 정보
		- 원자재 재고 관리
		- 원자재 입고 요청/입고 관리
		- 원자재 발주 요청 관리 (계획 포함)

	- 납품 관리
		- 완성차 정보
		- 납품사 정보
		- 납품사 연계 업체 정보
		- 납품사 창고 정보
		- 완제품 정보
		- 반제품 정보
		- 납품사 재고 관리
		- 완제품 입고요청/납품 관리

	-  생산 관리
		- 생산 공장 정보
		- 3개월 주문 계획 관리
		- 1주 주문 관리
		- 일별 생산 계획/실적 관리
		- 공정별 생산 계획/실적 관리
		- 원자재 사용 관리
		- 품질 관리

	- 재고 관리
		- 비엔케이 재고 창고 정보
		- 완제품 재고 관리
		- 반제품 재고 관리
